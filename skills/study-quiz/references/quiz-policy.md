# 퀴즈 출제 정책

> study-quiz 스킬에서 문제를 출제할 때 따르는 정책입니다.
> 이 정책의 목표는 **진짜 이해한 사람만 맞힐 수 있는 문제**를 만드는 것입니다.

---

## 기본 원칙

| # | 원칙 | 설명 |
|---|------|------|
| 1 | **힌트 제로** | 문제 지문이나 선택지에서 정답을 유추할 수 있는 단서를 주지 않음 |
| 2 | **오답의 매력** | 오답도 "그럴듯하게" — 피상적 이해로는 정답과 구분 불가 |
| 3 | **암기 아닌 이해** | 단순 정의 암기가 아니라 개념 적용 능력을 평가 |
| 4 | **공정한 위치** | 정답 위치(A/B/C/D)를 문제마다 랜덤으로 배치 |

## 난이도 배분

| 난이도 | 비율 | 기준 |
|--------|------|------|
| **기본** | 40% | 개념을 이해하면 바로 풀 수 있음 |
| **응용** | 40% | 두 개 이상의 개념을 조합해야 풀림 |
| **심화** | 20% | 함정이나 예외 케이스를 알아야 풀림 |

## 문제 작성 체크리스트

```
- [ ] 지문에 불필요한 힌트가 없는가?
- [ ] 오답이 "명백히 틀린" 것이 아닌가? (매력적 오답인지 확인)
- [ ] 정답이 가장 길거나 가장 구체적이지 않은가? (길이 편향 제거)
- [ ] "다음 중 옳지 않은 것"보다 "다음 중 올바른 것"을 우선 사용했는가?
- [ ] 정답 위치가 이전 문제와 연속으로 같지 않은가?
```

## 금지 패턴

| 패턴 | 문제점 | 대안 |
|------|--------|------|
| "다음 중 가장 적절한 것은?" (정답만 상세) | 길이로 정답 유추 가능 | 모든 선택지 비슷한 길이 유지 |
| A) 틀림, B) 틀림, C) 정답, D) "위 모두 해당" | D가 항상 정답 패턴 | "위 모두" 선택지 사용 자제 |
| "절대", "항상", "반드시" 포함 선택지 | 극단적 표현은 대체로 오답 | 중립적 표현 사용 |
| 이전 문제의 해설에서 다음 문제 정답 유추 | 문제 간 정보 유출 | 독립적 문제 설계 |

## 해설 작성 기준

| 항목 | 기준 |
|------|------|
| **정답 해설** | 왜 이것이 맞는지 1-2문장 |
| **오답 해설** | 가장 매력적인 오답 1개에 대해 왜 틀린지 설명 |
| **보충 설명** | 관련 개념을 연결하는 한 줄 (위키링크 포함) |
| **길이** | 전체 해설 3-5문장 이내 |

## 적응형 출제

사용자의 이전 결과에 따라 출제를 조정합니다:

| 이전 결과 | 조정 |
|----------|------|
| 전부 정답 | 심화 비율 30%로 상향 |
| 절반 이상 오답 | 기본 비율 50%로 상향 |
| 특정 개념 반복 오답 | 해당 개념 관련 문제 추가 출제 |

## 대화형 퀴즈 에티켓

1. 문제를 읽을 시간을 줌 — 출제 후 바로 답을 재촉하지 않음
2. 오답이어도 비난하지 않음 — "아쉽네요" 정도만
3. 3문제 연속 오답 시 — "개념노트를 한 번 보고 오시는 것도 좋아요" 제안
4. 사용자가 "모르겠어요"라고 하면 — 힌트 없이 정답과 해설 제공
