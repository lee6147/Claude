## 점수 산정

| 등급 | 점수 | 판정 |
|------|------|------|
| **A** | 90-100 | 바로 학습 시작 가능 |
| **B** | 80-89 | 소소한 보완 후 사용 가능 |
| **C** | 70-79 | 주요 항목 보완 필요 |
| **D** | 70 미만 | 재생성 권장 |

## 자주 빠뜨리는 항목

1. **정답 위치 편향** — A)에 정답이 몰리는 경우 → 랜덤 재배치
2. **해설 누락** — 접기 블록 없이 정답만 표시 → 해설 추가
3. **위키링크 오타** — `[[01-토픽]]` vs `[[01-토픽/개념노트]]` 불일치 → 통일
4. **빠른참조 미완성** — 빈 템플릿만 남아있는 경우 → 핵심 용어 채우기
