# Contributing

텐빌더 리소스를 개선해주셔서 감사합니다.

## 기여 방법

1. Fork → Branch → PR
2. 가이드 추가 시 기존 포맷을 따라주세요
3. 실제 프로덕션에서 검증된 패턴만 포함해주세요

## 스타일 가이드

- 한국어 기본, 코드/명령어는 영어
- 모든 예제는 **복사해서 바로 쓸 수 있어야** 합니다
- "왜?"를 설명하되, 불필요한 이론은 생략

## 이슈 & 질문

- 오류 제보: Issue 생성
- 질문: [뉴스레터](https://maily.so/tenbuilder) 구독 후 답장
